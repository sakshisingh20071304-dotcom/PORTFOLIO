# Sakshi Singh Portfolio

A premium, multi-page personal portfolio ecosystem built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Modern UI**: Glassmorphism, mesh backgrounds, and premium typography (Outfit).
- **Fully Responsive**: Optimized for desktop, tablet, and mobile.
- **Micro-animations**: Smooth transitions and interactive cursor glow.
- **Academic Focus**: Tailored for B.Tech CSE (AIML) student profile.
- **SEO Optimized**: Meta tags for search engines and social platforms.

## Deployment

### Option 1: Vercel (Recommended)
1. Initialize a Git repository: `git init`
2. Push to GitHub.
3. Import the project into Vercel. It will automatically detect the static files and apply the settings in `vercel.json`.

### Option 2: GitHub Pages
1. Push your code to a GitHub repository.
2. Go to `Settings > Pages`.
3. Select the branch (usually `main`) and root folder.

### Option 3: Netlify
- Drag and drop the project folder into the Netlify dashboard.
